Yak Shaving

6-28-2015

Shortly before my last post I started work on a clone of the famous Atari game Asteroids to learn the ropes of the SDL2 and OpenGL bindings on Common Lisp. While I was able to figure out how to get the game to a playable state by groking example code included in the git repositories of the aforementioned libraries however I soon found myself in a position where there were no examples to go by. To move foward with my original project, I decided to take on a few seemingly pointless activities to solve problems that will eventually solve problems I encountered with my original project, or as the Jargon File would call it Yak Shaving.
